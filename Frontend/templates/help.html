Social Distancing Detection
Home Signup/Login
If you want to upload a video for social distancing detection you need to first signup into the application.
Click on "Signup/Login", then there will appear a signup form enter your details and then it directs to Login page.
Enter your Login credentials and click the button "Login", it will directs to the Upload page.
There you can find "Choose File". Click on it. Select a video file from your system.
After selecting the video, click on buttons that you want to check either "Detection" or "Viloation".
If you have clicked the Detection button then there appears a python frame in the background which is output file will run showing the green bounding boxes over the people.press the "q" to end the output video python frame.
If you have clicked the Violation button then there appears a python frame in the background which is output file will run showing the violations with red bounding boxes over the people who are not maintaining the minimum diatnce and remaining will be shown with the green bounded boxes.
The video also consists of count of the violations and gives an voice alert as "Please maintaining Social Distance".press the "q" to end the output video python frame.
